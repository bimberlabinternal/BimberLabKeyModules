Assigning significance to genetics variants is often a challenge, and this is particularly problematic for model organisms like macaques, which have few sources of genomic annotations, relative to human or mouse. To overcome this, we developed a multipart process to annotate our data.

NHP data are aligned to the latest NCBI genomic build for the species. We use liftOver tools to lift variants to the human genome (GRCh37), with ~65% of variants typically being successfully lifted. These variants are annotated using a variety of human data sources, described below. These annotated variants are translated back to genomic coordinates of the NHP species, and all data sources merged. In addition to human annotations, SnpEff is used to annotate predicted impact on protein coding (run using the Ensembl gene build).

Our annotation sources are drawn from a number of databases and tools. A complete field-by-field list is shown in the table at the bottom of the page. Please note that in most cases the 'Data Source' column is a link to the primary data, which may have more information on that annotation.

  • Funcotator (which is a tool for annotation and an aggregation of data sources)
  • dbNSFP: Functional Predictions and Annotations for Human Non-synonymous and Splice Site SNVs
  • SnpEff: variant annotation and effect prediction tool
  • VariantAnnotation: a custom Funcotator-compatible set of annotation sources
